Parameter fitting

For this example, we try to fit a hyperbolic tangent to a set of data points. The equation of the curve is \(y = f(x)\) with \(f(x) = 2A\tanh(x/A) \). Here \(A\) is a parameter whose value we want to determine.

To judge how well the curve fits the data, we'll monitor the residual sum of squares (RSS), given by \[ \mathrm{RSS} = \sum_{i = 1}^N(y_i - f(x_i))^2, \] where the sum ranges over all data points \((x_i, y_i)\).

Instructions: Drag the curve up or down to change the fit. Better fits will result in a lower residual sum of squares (RSS). Hint: the optimal value for \(A\) is reasonably close to 50.
